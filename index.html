Securely manage your Nextcoins!

Nextcoinbox is a live CD set up to give you easy access to your Nextcoin wallet in a secured and isolated environment. When booting on this live CD (be it a virtual machine or a computer), the Nextcoin service gets started, and a browser opens to let you access the wallet. Closing the browser halt the virtual machine. This gives a very smooth experience, try it now!

Download! (216.2Mb)(md5: 9595bcfd59e59d871004328388cfd0c5)

Presentation

When you have downloaded the ISO, attach the ISO as the CD drive of a virtual machine, and start it. You then get a welcome page displayed in the browser, with a bookmark already setup to access the NextCoin wallet.

If the wallet does not load successfully, wait a bit and try again. It might be the Nextcoin service was not started yet.

This live CD can be used with any virtualisation technology, for example kvm, Xen, Vmware, Virtualbox, etc...

If you prefer, you can also burn it on a CD and boot any computer on it. This is your Nextcoin wallet on the go!

Based on Debian

The live CD was built on a plain Debian GNU/Linux distribution.
