Securely manage your Nextcoins!

Nextcoinbox is a virtual machine image (currently for Virtualbox) set up to give you easy access to your Nextcoin wallet in a secured and isolated environment. When booting the virtual machine, the Nextcoin service gets started, and a browser opens to let you access the wallet. Closing the browser halt the virtual machine. This gives a very smooth experience, try it now!

Download! (430Mb)

Presentation

When you have downloaded the image, simply import the appliance in VirtualBox (Menu File -> Import Appliance). When the import is done, simply start the newly created virtual machine. You then get a welcome page displayed in the browser, with a bookmark already setup to access the NextCoin wallet.

If the wallet does not load successfully, wait a bit and try again. It might be the Nextcoin service was not started yet.

Based on Debian

The virtual machine is running a plain Debian GNU/Linux distribution.

Securly manage and forge your Nextcoins

Running in an isolated environment, you limit the risk of malware getting access to your wallet.

Beta version

This is the first public release, and could contain bugs. You run this at your own risk.
